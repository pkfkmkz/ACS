Obsolete documents:

  • ANKA ACS Installation Report
    A short report on the upgrade of the ANKA Accelerator Control System to ACS 1.1.
    Notice that AKNA is not running under Linux, but on Microsoft (R) indows NT (TM) Workstation.
    The compiler used is MS Visual Studio 6.0 w/ Service Pack 5.
    The Windows porting of ACS is a completely independent work done by JSI on the ALMA ACS code. The Windows platform is not officially supported in the ALMA project.
