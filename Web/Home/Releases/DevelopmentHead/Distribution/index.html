ACS 2014.2 Distribution

Current official revision is: Linux ACS 2014.2 (RH 5.5 and RH6.5)

The ACS Distribution for Linux is available in two versions:

  • LGPL is publicly distributable according to the Lesser General Public Licence.
  • NO-LGPL is only distributable inside ALMA and the partner organizations and contains few tools and extensions that cannot be freely re-distributed

Online Documentation for this release is available here

ACS Download:

ftp server

ACS is released by means of tar files
  • ACS-YYYY_n-ExtProd-dddd-aaaa-oo.tar.gz: (where YYYY_n is the ACS release name; dddd is the date of the build, aaaa is the architecture like i686, oo is the operating system) this archive contains the external product needed to run ACS.
  • ACS-YYYY_n-dddd-aaaa-oo.tar.gz: contains ACS executables and configurations i.e. ACSSW and acsdat
To install ACS from the tars:
  1. login as almamgr
  2. go to the root foledr (/)
  3. unpack both the tars preserving the attributes (tar xpzf....)
  4. ensure to source the ACS bash profile (/alma/ACS-YYYY.n/ACSSW/config/.acs/.bash_profile.acs)
The URL of the ftp server is ftp.eso.org. Please drop us a email to get the username and password.

SVN mirror

ACS source and tools are kept under configuration control in a SVN archive.

All parts of the ACS archive publicly available as LGPL are mirrored in read only mode in a publicly accessible SVN repository, while development is done on a separate SVN, that contains also some developments and tools that we cannot publicly distribute. The mirror is updated once a week.
