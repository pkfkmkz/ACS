The INSTALL sub-directoriy contains a .acs directory that is normally not visible in the browser's directory listing.

It contains the following files:

INSTALL/.acs/.bash_profile.acs
INSTALL/.acs/emacs
INSTALL/.acs/emacs.local
INSTALL/.acs/xresources.emacs

The tar file INSTALL.tar.gz has the whole contents of the INSTALL directory and you can simply install and unpack it.
